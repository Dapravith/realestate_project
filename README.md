Getting started
  1. Clone the project
  $ git clone https://github.com/Dapravith/realestate_project.git
  2. Project setup
  
      $npm install
      
  3. Run the project
  
     $npm run dev
     
  # or to build
  
     $npm run build
  
  Deploy to Vercel
  
  Use the https://vercel.com/docs/cli#
  
  Vercel
